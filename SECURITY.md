# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in nri-supply-chain, please report it
responsibly. Do not open a public issue.

Please email the [SECURITY_CONTACTS](SECURITY_CONTACTS) directly or use
[GitHub's private vulnerability reporting](https://github.com/saschagrunert/nri-supply-chain/security/advisories/new).

## Supported Versions

Only the latest release is supported with security updates.
